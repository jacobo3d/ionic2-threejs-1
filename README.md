Ionic 2 Three js  App
=====================

This is Ionic 2 apps.

## Using this project

You'll need the Ionic CLI with support for v2 apps:

```bash
$ npm install -g ionic
```
then..
```bash
$ npm install
```
Then run:

```bash
$ ionic serve
```

Check Ionic
```bash
$ ionic info
```
More info on this can be found on the Ionic.
